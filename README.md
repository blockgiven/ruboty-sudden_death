# Ruboty::SuddenDeath

ruboty plugin for ＞突然の〜＜

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-sudden_death'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-sudden_death
